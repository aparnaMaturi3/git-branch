Commands used in git Branching

To list all the braches - git branch

To create a new branch and stay in the master branch - git branch new_branch_name

To create a new branch and switch to that branch - git checkout -b new_branch_name

To switch to different branch - git checkout branch_name

To merge a branch to master branch - checkout to master branch and then git merge branch_name_to_merge

To delete a branch - git branch -d branch_name
-d => shows details if we have unmerged files to be merged -D => deletes the branch even if we have unmerged files
